5. 実装フロー：状態と操作の分離

画像コメント導線

1一覧上部は入力カードを廃止し、「投資の話題を共有しよう」テキスト + 投稿ボタンの軽い入口にする。
2投稿ボタンで全画面寄りの投稿シートを開く。閉じる、本文、投稿ボタンは上部に固定し、本文入力領域は広く取る。
3関連ファンドと画像追加は下部ツールバーへ配置する。キーボード表示中はツールバーも一緒に上がる。
4画像追加後は本文下から底部にかけて横スクロールのサムネイルを表示し、各画像右上の × で削除できる。
5一覧カードは本文の後に画像グリッドを表示する。1枚は大きめ、2から3枚は均等グリッド、4枚以上は3列グリッド + 残数表示が自然。
6评论卡只显示回复数量，不在卡片内直接展示回复内容；点击卡片本体进入评论详情和返信一覧，列表内展示已有回复和图片。
7点击卡片上的回复数量按钮直接打开独立返信 Sheet；返信一覧里的“返信を書く”也打开同一个回复编辑页。
