Hotel booking redesign concept

典雅旅宿预约 UI 方向

保留老 App 的列表、地图、详情房型三段流程，但把视觉重心从功能堆叠改成“沉稳搜索、图片叙事、价格清晰、行动明确”。适配当前 StellaVia 深海军蓝、奶油底、金色高光与少量 teal 的品牌体系。

推荐使用的 skill

本次最合适的是静态 HTML/CSS 设计稿配合 Browser 预览验证。若后续要补充原创旅馆氛围图，再加 imagegen；HyperFrames 更适合视频稿，不适合作为普通 UI 设计文稿的主工具。

视觉基调

首页使用沉稳图片 hero 和浮层搜索卡，减少旧版大面积青色，按钮与价格强调回归品牌海军蓝和告警红，少量 teal 只用于筛选、地图价签和可操作状态。

信息架构

列表页突出目的地、日期、人数和推荐房源；地图页保留价格气泡和底部卡片；详情页把日期人数固定为强可见入口，房型卡片维持原有信息密度但降低杂乱感。
